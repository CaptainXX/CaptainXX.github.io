输入和输出

数据类型

  • 基本数据类型：

  • 整数、浮点、字符串、布尔值、空值（None）

  • 变量：声明变量时不用声明变量类型，变量类型不确定

  • 整除://,用/做除法时有小数

list

  • L = [1, 2, 3]

  • ‘len()’函数获取list元素的个数 z.B. len(L) = 3

  • ‘range(n,m)’函数产生包括n,不包括m的一个整数列，list(range(n,m))产生一个[n, n+1, n+2…m-1]的list

  • 通过索引访问每一个元素：L[0], L[1], L[2], L[-1], L[-2], L[-3]

  • 索引有越界问题

  • L.pop()方式删除末尾元素， L.pop(num)删除索引为num的元素

  • L.append(atom)方式在末尾添加一个元素

  • L.insert()方式插入一个元素，L.insert(num, atom)指定添加在索引为num的元素前

  • list中元素是可变的，通过索引可以直接对某个元素赋值

  • list中的元素可以是任何基本数据类型，或者是一个list，又或是一个tuple

tuple

  • t = (1, 2, 3, 4) 定义只有一个元素的tuple必须写做’t = (1,)’，(1)会直接作为数学中小括号运算

  • ‘len()’也对tuple有效

  • 也可以通过索引访问tuple中的元素

  • tuple中的元素定义后不可改变，但是如果tuple中有一个或几个list元素，则可以改变这些list中的元素

dict

  • 字典类型，采用键-值方式存储，键名不可变，值可以改变

条件判断

  • if 语句 : …

  • 如果if后面的语句结果为True，则执行冒号后面…处的语句

循环

  • for…in…循环
  • while循环
  • break,continue语句作用与C语音相同，配合判断语句使用

编码

  • python3 中以Unicode编码

  • ‘ord()’函数把字符转换为编码 ‘chr()’函数把编码转换为字符串
